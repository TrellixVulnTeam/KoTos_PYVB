How to enable GenreCover Support for Horizontal Confluence MOD:

Copy the thumbs from GenreCover-folder to \skin.moddedconfluence\media\moviegenres
Restart XBMC
ATTENTION: Icons will only display if they are named like Genre. An Icon called "scifi.jpg" wont't
show up for Genre "Science-Fiction". You know what I mean...

Kopiert die Cover von "GenreCover" nach \skin.moddedconfluence\media\moviegenres
XBMC neu starten
ACHTUNG: Die Cover werden nur angezeigt wenn Sie den gleichen Namen wie das Genre haben.
Das Cover "scifi.jpg" wird für ein Genre "Science-Fiction" nicht angezeigt...Ihr versteht schon...
